üìò Project Introduction

This project is a browser-based interactive tool designed to explore and compare news outlets based on bias, factuality, and country of origin. The primary goal is to help users develop media literacy by providing an intuitive interface to filter and visualize media sources through engaging charts and interactive controls.

In today's information-rich environment, understanding the political leanings and credibility of news sources is crucial for making informed decisions. This tool democratizes access to media analysis by presenting complex data in an easily digestible format.
